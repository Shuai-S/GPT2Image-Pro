## 快速开始

1. 在控制台创建 **API Key**（外部 API 需 Starter 及以上套餐）。
2. 读 **[系统文档](/docs/system)** 的「通用规则」与鉴权说明。
3. 图像走 `POST /v1/images/generations`；**视频是长任务、强烈建议异步**：`POST /v1/videos/generations` 带 `async:true` 立即拿 `task_...`，再轮询 `GET /v1/videos/{id}` 或用 `callback_url` 回调。
