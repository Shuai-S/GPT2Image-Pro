这份手册面向第一次进入网站的用户，按真实使用路径说明每个入口能做什么、什么时候该选它、以及应该点哪里。

## 快速开始

1. 点击顶部导航的 **控制台**。如果还没登录，先按页面提示注册或登录。
2. 进入控制台后点击 **开始创作**，或在左侧菜单选择 **创作**。
3. 在创作页顶部选择模式：**文生图**、**图生图**、**对话**、**Agent**、**瀑布流** 或 **视频**。
4. 先用默认参数生成一张图。确认流程跑通后，再调整模型、尺寸、质量、背景、后端分组等选项。
5. 生成结果可直接 **下载**，也会保存到 **图库** 和 **历史记录**。

## 先选哪个功能

| 你想做什么 | 选择哪个入口 | 怎么点 |
| --- | --- | --- |
| 只用文字生成图片 | **创作 → 文生图** | 输入提示词，选择 **单提示词**，点 **生成**。 |
| 一次生成多条不同提示词 | **创作 → 文生图** | 切到 **逐行批量**，每行写一个提示词，点 **生成逐行批量**。 |
| 改一张已有图片 | **创作 → 图生图** | 点 **上传图片**，写清楚要改哪里，点 **生成**。 |
| 局部重绘或替换元素 | **创作 → 图生图** | 上传源图后点 **绘制蒙版** 或 **上传蒙版**，保存蒙版后生成。 |
| 像聊天一样连续改图 | **创作 → 对话** | 在输入框描述需求，可添加参考图，然后发送。 |
| 让 AI 自己搜索、拆解、迭代 | **创作 → Agent** | 选择轮数，必要时勾选 **强制**，输入目标后发送。 |
| 持续批量刷灵感图 | **创作 → 瀑布流** | 设置 **每批** 张数，输入一个主提示词，点发送按钮。 |
| 生成视频 | **创作 → 视频** | 选择视频模型、时长、比例和分辨率，输入提示词后提交。 |
| 管理成品图片和视频 | **图库** | 用 **成品**、**Agent 草稿**、**上传图**、**视频** 标签切换。 |
| 查看失败、处理中和扣费明细 | **历史记录** | 打开记录查看状态、错误、积分消耗和参考图。 |
| 查看余额、购买积分或订阅 | **账单与用量** | 在 **用量** 看余额，在 **账单** 管理套餐。 |
| 给程序或第三方工具调用 | **外接 API** | 创建 API Key，再按 **系统文档** 调用接口。 |

## 创作页通用操作

进入 **创作** 后，页面左侧通常是输入区和结果区，右侧或顶部是参数区。不同模式会显示不同控件，但选择原则一致：

- **后端分组**：普通用户保持默认即可。只有你知道某个分组更适合某类任务时再切换；切换后，本次创作会按该分组的倍率计费。
- **图片模型**：默认模型适合大多数图片。需要特定模型风格或 Firefly 路线时，再从模型下拉框切换。
- **尺寸**：不确定就选 **Auto**。做头像、商品图、海报、横幅时再改成对应比例或自定义尺寸。
- **质量**：草稿和试提示词选低或默认；最终交付图再提高质量。
- **背景**：需要透明 PNG、贴纸、Logo 素材时选择透明；普通照片、海报、插画选自动或不透明。
- **高清修复**：结果尺寸偏小或细节不够时开启；它可能更慢，也可能产生额外消耗。
- **分块修复**：图片文字、局部细节需要加强时再开；适合最终稿，不适合每次试稿都开。
- **提示词优化**：想让系统帮你补充细节时开启；如果必须严格保留原话，就关闭或选择尽量减少改写。

## 文生图

**适合**：从零生成头像、海报、插画、商品概念图、社交媒体配图。

使用步骤：

1. 进入 **创作 → 文生图**。
2. 在输入框写清楚主体、场景、风格、构图、颜色和用途。
3. 选择 **单提示词**。
4. 按需调整 **图片模型**、**尺寸**、**质量** 和 **背景**。
5. 点击 **生成**。
6. 结果满意就点 **下载**；想继续改，点 **继续编辑** 或去 **图生图** 上传结果。

提示词建议：

- 好的写法：主体 + 场景 + 风格 + 构图 + 光线 + 用途。
- 示例：`一张科技产品发布会主视觉，银色耳机悬浮在深色舞台中央，蓝白灯光，高级商业摄影，16:9 横幅。`
- 如果第一张不满意，优先改提示词里的主体和风格；最后再改高级参数。

## 逐行批量

**适合**：你已经有多条方向，想一次生成多张不同图。

使用步骤：

1. 进入 **创作 → 文生图**。
2. 切换到 **逐行批量**。
3. 每行输入一个完整提示词。
4. 查看页面上的 **提示词行数** 和 **总图片数**。
5. 设置模型、尺寸和质量后，点击 **生成逐行批量**。

选择建议：

- 每行代表一张图，不要把多个需求挤在同一行。
- 批量试风格时保持尺寸一致，方便对比。
- 积分会按实际生成张数消耗，生成前先看余额。

## 图生图

**适合**：保留原图主体，改风格、换背景、补细节、做局部重绘。

使用步骤：

1. 进入 **创作 → 图生图**。
2. 点击 **上传图片**，上传 PNG、JPEG 或 WebP。
3. 在文本框写明要怎么改，例如“保留人物姿势，把背景换成夜晚城市街道”。
4. 如需局部修改，点击 **绘制蒙版**，在第一张源图上涂出需要改的区域，然后点 **保存蒙版**。
5. 也可以点击 **上传蒙版**，上传一张 PNG 蒙版。
6. 设置尺寸。想尽量保持源图比例时，用自动或接近源图的比例。
7. 点击 **生成**。

蒙版选择：

- 只改局部：用 **绘制蒙版**，涂需要变化的区域。
- 已有专业遮罩：用 **上传蒙版**。
- 整张图重风格化：不要加蒙版，直接描述整体变化。

## 对话

**适合**：边聊边改、连续迭代、多轮确认方向。

使用步骤：

1. 进入 **创作 → 对话**。
2. 在底部输入框描述你要的图片。
3. 需要参考图时，点击输入框旁的添加图片按钮，或粘贴图片。
4. 需要引用历史图时，输入 `@` 并选择要引用的图片。
5. 按需设置 **尺寸**、**背景**、**图片模型**、**高清修复**。
6. 发送后等待结果。
7. 对结果继续提要求，例如“保持构图，把配色改成暖色”“再生成一个更极简的版本”。

选择建议：

- 想快速出一张图：用 **文生图**。
- 想持续微调同一个方向：用 **对话**。
- 想让系统主动拆任务、搜索资料、连续迭代：用 **Agent**。

## Agent

**适合**：复杂创作目标，例如品牌主视觉、角色设定、多轮迭代、需要搜索资料或读取附件的任务。

使用步骤：

1. 进入 **创作 → Agent**。
2. 写清楚最终目标、风格约束、不能出现的内容、交付用途。
3. 选择轮数。一般先选 2 到 3 轮；复杂任务再提高。
4. 如果必须跑满所选轮数，勾选 **强制**。
5. 如需 PSD 分层，勾选 **打散元素生成 PSD**。
6. 发送后观察任务卡。任务卡会展示搜索、工具调用、生图和继续迭代过程。
7. 结果出来后，可在对话里继续要求它修改，也可去 **图库 → Agent 草稿** 查看中间版本。

选择建议：

- 初稿探索：不用勾选 **强制**，让 Agent 自己判断是否继续。
- 必须多轮打磨：勾选 **强制** 并提高轮数。
- 需要后期设计编辑：勾选 **打散元素生成 PSD**。
- 只想便宜快速试一张：不要用 Agent，用文生图或对话。

## 瀑布流

**适合**：一个主题连续产出大量灵感图，用来挑选方向。

使用步骤：

1. 进入 **创作 → 瀑布流**。
2. 在顶部选择 **每批** 张数。数值越大，并发越高，消耗越快。
3. 设置尺寸、质量、背景和修复选项。
4. 在中间输入一个主提示词。
5. 点击发送按钮开始瀑布流。
6. 运行中看顶部状态：已发送、成功、失败、进行中。
7. 看到满意结果后下载，或把它作为参考图继续编辑。

选择建议：

- 找灵感：每批选择较小数值，先看方向。
- 批量生产：确认提示词稳定后再提高每批张数。
- 控制成本：关闭不必要的高清修复和分块修复。

## 视频

**适合**：将文字或画面创意扩展成短视频。

使用步骤：

1. 进入 **创作 → 视频**。
2. 选择视频模型。
3. 选择时长、比例和分辨率。
4. 输入视频提示词，说明主体动作、镜头运动、风格和画面节奏。
5. 提交任务后等待生成。
6. 完成的视频会进入 **图库 → 视频**。

选择建议：

- 先短时长测试动作和风格，再生成更长版本。
- 分辨率越高、时长越长，通常越慢且消耗更多。
- 如果任务较久，不要重复提交；去 **历史记录** 或 **图库 → 视频** 查看状态。

## 图库

**图库** 用来管理已经完成的图片和视频。

常用标签：

- **成品**：最终完成并可下载的图片。
- **Agent 草稿**：Agent 多轮过程中产生的中间图。
- **上传图**：你在图生图、对话、瀑布流中上传过的参考图。
- **视频**：已完成的视频结果。

常用操作：

- 点击作品可预览大图或视频。
- 使用 **下载** 保存到本地。
- 需要继续改图时，把作品作为参考图带回创作页。
- 分层作品可用于后续 PSD 导出或设计编辑。

## 历史记录

**历史记录** 会显示所有任务，包括成功、失败和处理中。

你可以在这里看：

- 任务状态。
- 使用的模型和尺寸。
- 消耗的积分。
- 失败原因。
- 参考图和提示词。

使用建议：

- 找不到刚才的结果：先看 **历史记录**，再看 **图库**。
- 任务失败：打开失败记录看错误原因，再决定是否重试。
- 对账积分：查看记录里的积分消耗明细。

## 账单与用量

在 **账单与用量** 可以管理套餐、积分和交易记录。

常用操作：

1. 打开 **账单与用量**。
2. 在 **用量** 查看可用积分、即将过期积分和交易记录。
3. 积分不足时，点击 **按量付费** 购买一次性积分，或点击 **查看订阅套餐** 升级套餐。
4. 在 **账单** 查看当前套餐、续费日期和账单历史。

选择建议：

- 偶尔创作：购买一次性积分。
- 经常创作：选择订阅套餐，获得每月积分和更多功能。
- 需要对话、API 或更高分辨率：根据定价页的套餐权益升级。

## 外接 API

**外接 API** 面向开发者或需要把生图能力接入其他工具的用户。

使用步骤：

1. 进入 **外接 API**。
2. 点击 **创建 API Key**。
3. 给 Key 命名，例如 `production` 或 `test`。
4. 可选填写 **积分限额**，限制这个 Key 最多消耗多少积分。
5. 可选选择 **审核级别** 和 **后端分组**。
6. 创建后立即复制 Key。它只完整显示一次。
7. 按 **[系统文档](/docs/system)** 里的 Base URL、鉴权和接口示例调用。

选择建议：

- 只是网页创作：不需要 API Key。
- 给脚本、工作流、第三方站点调用：创建 API Key。
- 担心误用或成本失控：一定设置积分限额。
- 多个项目共用账号：给每个项目单独创建 Key，方便撤销和追踪。

## 设置

在 **设置** 中管理账号和高级偏好。

常用区域：

- **账户**：修改姓名、头像等基本信息。
- **安全**：修改密码。
- **账单与用量**：查看套餐、积分和交易记录。
- **外接 API**：创建和管理 API Key。
- **后端**：选择网页端创作默认使用的系统后端账号池分组。
- **高级**：配置自定义 OpenAI 兼容 API 端点。

选择建议：

- 不懂后端分组：保持默认。
- 想网页端优先使用某个系统分组：到 **设置 → 后端** 修改。
- 有自己的 OpenAI 兼容 API，且不想消耗平台积分：到 **设置 → 高级** 启用自定义 API。
- 修改 API 后先点测试出图，确认可用再正式使用。

## 顶部导航和页面按钮

- **创作**：进入图片和视频生成工作区。
- **定价**：查看套餐、积分、功能权益和升级入口。
- **文档**：打开本使用手册和高级 API 文档。
- **控制台**：查看余额、最近创作和常用入口。
- **语言图标**：切换界面语言。
- **月亮图标**：切换深色或浅色主题。
- **头像**：进入账号相关菜单。

## 常见选择

| 场景 | 推荐选择 |
| --- | --- |
| 第一次试用 | 文生图 + 默认参数 + 单提示词。 |
| 做最终海报 | 文生图或 Agent，尺寸选目标比例，最终稿再开高清修复。 |
| 改商品背景 | 图生图，上传商品图，必要时用蒙版保护商品主体。 |
| 角色连续迭代 | 对话，持续引用上一张图修改。 |
| 复杂商业视觉 | Agent，设置 2 到 4 轮，必要时强制跑满。 |
| 快速找风格 | 瀑布流，每批少量开始，挑中方向后再批量。 |
| 控制成本 | 默认质量、Auto 尺寸、少开修复、先小批量测试。 |
| 开发集成 | 外接 API，设置积分限额，读系统文档。 |
