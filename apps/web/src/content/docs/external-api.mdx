本站提供 OpenAI 兼容的外部 API，可用 API Key 直接生成图像与视频。所有端点在
`/v1/*` 下（亦有等价镜像 `/api/v1/*`）。

## 鉴权

- 在 控制台 → API Key 处创建密钥，请求头携带 `Authorization: Bearer <API_KEY>`。
- 外部 API 是 **Starter 及以上套餐**的能力；免费套餐调用返回 `403 insufficient_plan`。
- 浏览器跨域（CORS）默认开放，可在后台「外部 API 跨域」开关控制。

基础地址（base URL）：`https://gpt2image.superapi.buzz/v1`（下文示例用 `$BASE`）。

```bash
export BASE="https://gpt2image.superapi.buzz/v1"
export KEY="sk-..."   # 你的 API Key
```

## 通用约定

- 请求体为 JSON（图像编辑额外支持 `multipart/form-data`，见下）。
- 响应为 OpenAI 风格；产物默认以**签名 URL**返回（`data[].url`），URL 短时有效。
- **积分**：每次成功生成按尺寸/时长 × 倍率扣费，响应含 `credits_consumed`。失败一般不扣分；
  审核拦截、格式不支持等属用户错误，返回 4xx。
- **错误格式**（HTTP 4xx/5xx），OpenAI 风格信封：

```json
{ "error": { "message": "...", "type": "invalid_request_error", "code": "..." } }
```

- 蛇形/驼峰参数大多两种都接受（如 `callback_url` 与 `callbackUrl`、`response_format`）。

## 图像

### POST /v1/images/generations — 文生图

常用参数（`prompt` 必填，其余可选）：

| 参数 | 类型 | 说明 |
|---|---|---|
| `prompt` | string | 提示词（必填） |
| `model` | string | 模型 id；省略走默认。firefly 家族 id（如 `firefly-nano-banana-pro`）或 `gpt-image`。见 `/v1/models` |
| `n` | int | 张数（1 起，上限随套餐） |
| `size` | string | 如 `1024x1024`、`2048x2048`、`auto` |
| `quality` | enum | `auto` / `low` / `medium` / `high` |
| `response_format` | enum | `url`（默认）/ `b64_json` |
| `output_format` | enum | `png` / `jpeg` / `webp` |
| `background` | enum | `transparent` / `opaque` / `auto` |
| `moderation` | enum | `auto`（默认）/ `low` |
| `stream` | bool | SSE 流式（含 partial 进度，见下） |
| `async` | bool | 异步：立即返回 `task_...`，凭任务查询接口轮询 |
| `callback_url` | string(url) | 异步完成回调地址 |
| `force_firefly` | bool | 强制本次路由到 Adobe（firefly）后端 |

```bash
curl -s $BASE/images/generations \
  -H "Authorization: Bearer $KEY" -H "Content-Type: application/json" \
  -d '{"prompt":"a calico cat, studio light","model":"gpt-image","size":"1024x1024"}'
```

响应（同步）：

```json
{
  "created": 1782115471,
  "data": [{ "url": "https://.../api/storage/generations/.../x.png?sig=...&exp=...",
             "revised_prompt": "..." }],
  "generation_id": "vKOqsNq4QZOb5R7upgKzS",
  "credits_consumed": 9.06
}
```

`response_format: "b64_json"` 时 `data[].b64_json` 返回 base64。

### POST /v1/images/edits — 图生图 / 编辑

两种请求体：

- `multipart/form-data`：字段 `image`（一或多个文件）、可选 `mask`（PNG）、`prompt`、`model`、`size` 等。
- `application/json`：`image` 为 base64 data URL 或图片 URL 数组，`mask_url` 可选，其余同上。

```bash
curl -s $BASE/images/edits \
  -H "Authorization: Bearer $KEY" \
  -F image=@cat.png -F prompt="put a hat on the cat" -F size=1024x1024
```

响应同 `generations`。

### `GET /v1/images/{id}` — 查询图像任务 / 复查

`{id}` 可为异步返回的 `task_...`，或同步响应里的 `generation_id`（DB 持久，跨重启/多实例可查）。

```json
{
  "id": "vKOqsNq4QZOb5R7upgKzS",
  "object": "image",
  "model": "gpt-image",
  "status": "completed",
  "created_at": "2026-06-22T08:04:31.109Z",
  "completed_at": "2026-06-22T08:05:38.633Z",
  "image_url": "https://.../x.png?sig=...&exp=...",
  "data": [{ "url": "https://.../x.png?sig=...&exp=..." }],
  "credits_consumed": 9.06
}
```

`status`：`processing` / `pending` / `completed` / `failed`（失败含 `error`）。

## 视频

视频是长任务，**建议异步**（`async: true`）。模型 id 形如
`firefly-<family>-<dur>s-<ratio>[-<res>]`（如 `firefly-sora2-4s-16x9`），可从 `/v1/models` 获取。

### POST /v1/videos/generations

| 参数 | 类型 | 说明 |
|---|---|---|
| `prompt` | string | 提示词（必填） |
| `model` | string | firefly 视频模型 id（必填） |
| `negative_prompt` | string | 反向提示词（可选） |
| `image` | string[] | 图生视频输入图，base64 data URL，最多 3 张（首帧/参考） |
| `async` | bool | 异步：立即返回 `task_...`（强烈建议） |
| `callback_url` | string(url) | 异步完成回调（POST 任务对象） |

异步提交（推荐）：

```bash
curl -s $BASE/videos/generations \
  -H "Authorization: Bearer $KEY" -H "Content-Type: application/json" \
  -d '{"prompt":"a drone shot over a beach","model":"firefly-sora2-4s-16x9","async":true}'
```

异步立即返回任务：

```json
{ "id": "task_5d936d7e...", "object": "image.generation", "model": "firefly-sora2-4s-16x9", "status": "processing" }
```

随后凭 `id`（`task_...` 或 `generation_id`）轮询 `GET /v1/videos/{id}`。
同步模式（不传 `async`）会用 keep-alive 撑住连接，跑完直接返回，响应含
`data[].url`、`generation_id`、`credits_consumed`。

### `GET /v1/videos/{id}` — 查询视频任务

```json
{
  "id": "task_5d936d7e...",
  "object": "video",
  "model": "firefly-sora2-4s-16x9",
  "status": "completed",
  "duration_seconds": 4,
  "created_at": "2026-06-22T08:04:31.109Z",
  "completed_at": "2026-06-22T08:05:38.633Z",
  "video_url": "https://.../x.mp4?sig=...&exp=...",
  "data": [{ "url": "https://.../x.mp4?sig=...&exp=..." }],
  "credits_consumed": 60
}
```

`status` 同图像；未完成时 `object` 为 `video.generation`，无 `*_url`。
`callback_url` 会在完成/失败时收到一次该任务对象的 POST。

## 其它端点

| 端点 | 说明 |
|---|---|
| `GET /v1/models` | 列出可用模型（图像家族 + firefly 视频家族）：`{ object:"list", data:[{id, object:"model"}] }`，用于发现 model id |
| `POST /v1/chat/completions` | OpenAI Chat 兼容的对话生图（chat-to-image） |
| `POST /v1/responses` | OpenAI Responses 兼容的图像生成 |
| `GET /v1/credits` | 查询当前 API Key / 账号积分余额 |
| `POST /v1/agents/images` | Agent 模式多轮生图 |

## 错误码速查

| HTTP | code | 含义 |
|---|---|---|
| 401 | `invalid_api_key` | Key 缺失/无效 |
| 403 | `insufficient_plan` | 套餐不含外部 API（需 Starter+） |
| 400 | `invalid_request_error` | 参数非法（尺寸/模型/JSON 等） |
| 400 | `image_generation_user_error` | 用户输入问题（格式不支持如 mpo/avif、尺寸不符、审核拦截等），不扣分 |
| 5xx | `upstream_error` | 上游/平台暂不可用，可重试 |
